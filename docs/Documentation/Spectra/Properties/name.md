# `spectra.name`

This property gives access to the name associated with a spectra instance.

### Getter

#### Returns
{: .no_toc }
- `str`

### Setter

#### Parameters
{: .no_toc }
- name (`str` or anything that can be converted to `str`)

### Deleter

Resets the name to an empty `str` (`''`).
