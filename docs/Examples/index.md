Some example files are present in the repertory `Examples`.
You can copy them in your working directory to test them.

`main.py` and `constants.py` works together: the second provide useful constants such as data access path, while the second contains the main functions (preprocessing, plotting, ...).

They can be used in interactive mode, by importing `main.py`:

```python
from main import *
```
