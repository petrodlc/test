## Dependencies

### For python only

Dependencies required:
- python (tested only with python 3.14.5)
- numpy  (tested with v2.4.6)
- scipy (tested with v1.17.1)
- pathlib (should be installed along with python)

Recommended:
- matplotlib, as no plotting utility is provided
